# Emiakpo Global Ventures – Land_for_sale

A modern full-stack real estate platform developed by **Emiakpo Global Ventures** to help buyers discover, evaluate, and purchase lands with confidence.

The platform provides accurate property information, plot pricing, location details, community insights, nearby landmarks, and interactive maps to simplify the land acquisition process.

---

## About Emiakpo Global Ventures

Emiakpo Global Ventures is a real estate company committed to helping individuals and organizations secure valuable land investments through transparency, accessibility, and technology-driven solutions.

---

## Project Overview

Land_for_sale is a real estate web application designed to showcase available lands and properties while providing buyers with all the information they need before making a purchase decision.

### Key Objectives

- Simplify land discovery
- Improve transparency in property sales
- Provide accurate location information
- Showcase surrounding communities and landmarks
- Connect buyers directly with sellers

## Installation Guide

### Clone Repository

```bash
git clone https://github.com/yourusername/Land_for_sale.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file inside the server directory.

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
